# Style Keywords Reference

Aesthetic keywords organized by category. Include these in prompts to steer visual output. Most effective when placed near the end of the prompt as style modifiers.

## Cinematic Looks

| Keyword | Effect |
|---------|--------|
| "cinematic" | General film quality boost, letterbox feel |
| "shot on 35mm film" | Film grain, warm tones, classic cinema |
| "shot on 16mm" | Indie film grain, vintage, more texture |
| "anamorphic widescreen" | Horizontal lens flare, oval bokeh, 2.39:1 feel |
| "anamorphic lens" | Same as above, slightly less explicit |
| "IMAX" | Sharp, immersive, large-format feel |
| "Kodak 2383" | Warm film stock reference, classic print look |
| "Kodak 5219" | Tungsten-balanced stock, indoor warmth |
| "film noir" | High contrast B&W, dramatic shadows |
| "shallow depth of field" | Subject isolation, bokeh background |
| "deep focus" | Everything sharp, Citizen Kane style |
| "lens flare" | Light artifacts for realism and warmth |
| "film grain" | Texture, analog feel, organic |
| "color grading: teal and orange" | Hollywood blockbuster palette |
| "bleach bypass" | Desaturated, high contrast, metallic |

## Combining Keywords

### Effective Combinations
```
"Shot on 35mm film, golden hour backlighting, shallow depth of field,
muted earth tones with warm amber highlights"
```

```
"Neon-lit cyberpunk alley, wet surfaces reflecting colored light,
volumetric fog, anamorphic lens flare, high contrast"
```

### Anti-Patterns
- Don't mix contradictory styles: "golden hour" + "harsh fluorescent"
- Don't stack too many: 3-5 style keywords is optimal
- Don't use vague modifiers: "beautiful" adds nothing, "volumetric golden hour backlighting" adds everything
- Don't mix eras without intention: "1940s noir" + "cyberpunk" works, but be deliberate
